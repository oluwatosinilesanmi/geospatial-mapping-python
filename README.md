# Python GIS Mapping Course

This repository contains materials for learning how to perform geospatial mapping using Python.

The course demonstrates how to:

• Read shapefiles using GeoPandas  
• Join attribute tables to spatial data  
• Clean spatial datasets  
• Create choropleth maps  
• Add basemaps with Contextily  
• Label geographic features  
• Export professional cartographic maps  

---

## Course Topics

1. Installing Python GIS libraries
2. Reading shapefiles
3. Loading CSV data
4. Joining tables to spatial data
5. Data cleaning
6. Creating choropleth maps
7. Adding basemaps
8. Cartographic design
9. Labeling map features
10. Exporting maps

---

## Required Libraries

Install dependencies:

pip install geopandas pandas matplotlib contextily mapclassify

or

pip install -r requirements.txt
